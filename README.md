# DCP Camera Profile Spoofer
Allows switching between multiple Adobe Camera Match profiles by changing the camera name.

For example, it enables users to activate new color profiles—such as CineLike A2, L Classic, and L Classic Gold (originally only for the S1 II and L10 models)—on the S5 II and older. It is also compatible with cameras from Sony, Canon, Nikon, and other brands. If you want to maximize color advantages, ensure that both cameras use the same CMOS sensor. (for example, a7iii to s5,imx410 to imx 410)
